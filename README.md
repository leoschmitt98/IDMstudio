
  # Landing page premium IDM Studio

  This is a code bundle for Landing page premium IDM Studio. The original project is available at https://www.figma.com/design/gmI0MblDhv442xFEjjWg5n/Landing-page-premium-IDM-Studio.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

Run `npm test` to execute the automated tests.

Set `VITE_WHATSAPP_NUMBER` in your environment (e.g. `.env`) to configure the WhatsApp destination used by CTA buttons.
